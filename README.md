# Beauty (Saloon App)
A Saloon app to book appointments online with firebase

# Features
- SignUp With Email, Password
- SignIn with Email, Password
- SignIn With Google
- SignIn with Phone Number
- Book Appointments Online
- Pay Online
- Track Your Appointments
